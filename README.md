## Sobre o Jogo

*Um joguinho de nave construido em C usando a biblioteca Allegro.*

**Gênero:** Arcade  
**Plataformas:** Linux  
**Status:** Em desenvolvimento / Versão 1.0 lançada

---

## Como Jogar

### Objetivo
 O jogo é composto por três fases, não rele nas naves, evite as balas.

### Controles

| Ação               | Tecla |
|--------------------|--------------------|
| Movimentar para cima | ⇧ |
| Movimentar para baixo | ⇩ |
| Movimentar para a direita | ⇨ |
| Movimentar para a esquerda | ⇦ |
| Atirar             | Espaço |


> 💡 **Dica:** *Os inimigos se movimentam no mapa, eventualmente podem se "teleportar" cuidado!*
---

## Como Executar

### Pré-requisitos

- **Bibliotecas necessárias:** (liballegro-audio5-dev liballegro-dialog5-dev, 
                                liballegro-acodec5-dev liballegro-video5-dev, 
                                liballegro-physfs5-dev)


# Compile com Make
make

# Execute
./jogo
